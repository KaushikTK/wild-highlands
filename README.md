## 🦁 Wild Highlands: Survival FPS

A high-octane, browser-based survival first-person shooter built with Three.js. Navigate the sunny highlands, fend off aggressive predators, and survive as long as you can against the encroaching wildlife.

🚀 Play Now

Live Demo: https://kaushiktk.github.io/wild-highlands/


## 🎮 How to Play

#### 💻 Desktop Controls

WASD: Movement

Mouse: Look around (Pointer Lock enabled)

Left Click: Fire Weapon

Right Click: Aim Down Sights (ADS)

G: Toss Fragment Grenade

#### 📱 Mobile Controls

Left Virtual Joystick: Move character

Right Touch Area: Swipe to look/aim

FIRE Button: Shoot

ADS Button: Toggle zoom/precision mode

G Button: Deploy Grenade

Note: For the best experience, play in Landscape Mode. The game will automatically attempt to enter fullscreen.

## 🛠️ Technical Features

Engine: Three.js (WebGL)

Procedural Terrain: Dynamic heightmap generation using sine/cosine wave layering for realistic hills and valleys.

Physics Engine: Custom-built ray-casting and distance-based collision detection for bullets, grenades, and enemy AI.

Responsive Design:

Full mobile support with custom touch UI.

Automatic orientation locking and Fullscreen API integration.

## Graphics:

Real-time shadows (optimized for desktop).

Muzzle flashes, explosions, and fog-based depth cueing.

Animated 3D models using GLTFLoader and AnimationMixer.
